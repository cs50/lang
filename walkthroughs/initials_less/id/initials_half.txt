ZAMYLA CHAN: Dalam inisial masalah, kami akan meminta pengguna untuk nama mereka dan kemudian mencetak inisial dari nama itu. Untuk meminta pengguna untuk nama, kita akan menggunakan fungsi get_string ditemukan di CS 50 perpustakaan. 

Anda diperbolehkan untuk membuat dua assumptions-- satu, bahwa string akan terdiri hanya huruf dan spaces-- sehingga tidak ada nomor, tidak ada karakter lain. Anda juga diperbolehkan untuk berasumsi bahwa hanya ada akan menjadi satu ruang antara huruf, antara nama. 

Jadi setelah Anda memiliki string yang, kemudian mari kita pergi untuk mencetak inisial. Dalam contoh ini, mengatakan saya yang menjadi pengguna sempurna. Maka saya akan ketik nama saya dengan Z modal dan modal C, dengan satu ruang di antara. Jadi program anda akan menghitung huruf pertama dan kemudian huruf pertama setelah ruang, dan print out ZC. 

Tapi misalnya, mengatakan Mile Pisang adalah di keyboard. Yah, ia mungkin sedikit ceroboh daripada aku am dan hanya ketik huruf kecil surat dan mencakup ruang ekstra di depan nama pertamanya. Tapi program anda harus memiliki output yang persis sama, dengan modal MB tanpa spasi. 

Akhirnya, setiap kali Anda sudah menentukan karakter untuk mencetak, pastikan bahwa Anda mencetak mereka sebagai huruf besar menggunakan fungsi toupper. 

Nama saya ZC, dan ini adalah inisial.
