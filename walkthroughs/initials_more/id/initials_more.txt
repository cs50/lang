ZAMYLA CHAN: Dalam inisial masalah, kami akan meminta pengguna untuk mereka nama dan kemudian mencetak inisial mereka. Agar meminta nama, kita akan menggunakan mendapatkan fungsi string yang ditemukan di CS 50 perpustakaan, dan Anda diperbolehkan untuk berasumsi bahwa string ini hanya akan berisi huruf dan spaces-- sehingga tidak ada nomor dan tidak ada karakter lain. Kemudian, setelah Anda memiliki string, Anda akan iterate melalui. Jika, mengatakan, saya menggunakan program ini, maka saya mungkin ketik nama saya perfectly-- Zamyla Chan dengan Z modal di awal, spasi, dan kemudian modal C untuk memulai nama terakhir saya. Jadi Anda hanya akan iterate melalui string, mendapatkan karakter pertama, dan kemudian karakter pertama setelah spasi dan mencetak mereka keluar. 

Tapi mengatakan Anda memiliki sedikit sloppier pengguna, Milo Banana, yang tidak ketik huruf besar dan mungkin termasuk beberapa ruang asing karakter. Nah, maka ketika Anda iterate melalui string, Anda harus memeriksa untuk memastikan bahwa Anda tidak mencetak spasi, melainkan, nya tepat inisial, modal M, modal B. Untuk mencetak huruf kapital, Anda dapat menggunakan fungsi toupper. Dan dengan itu, Anda selesai masalahnya.
